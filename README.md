# stampee

## ⛳Ground Rule
- 10시 스크럼 회의
- 4시 전 완성된 파트 pr올리기
- 노션 상황 공유에 트러블 슈팅 기록하기
- 카톡보면 체크표시 or 답장 무조건하기
- 기능 단위 커밋 올리기
- 주말 오후 2시까지 출석


## 🖥️Developer Rule
- IDE는 인텔리제이로
- 구글 스타일 컨벤션으로 인텔리제이 포매터 설정하기

#### ✏️PR Template
```
## pr타이틀 //지워서 보내주세요

## 변경사항

## 구현사항

## 고민사항
```

#### 👿Commit Convention
```
- [FEAT]: 기능 추가
- [DOCS]: 문서 작성
- [REFACTOR]: 리펙토링
- [FIX]: 오류 수정
- [TEST]: 테스트 코드 추가
- [STYLE]: 네이밍 수정
```
